# The Village

How to Edit The Village Data

## Step 1: Open the Web Editor

1. Go to The Village repository on GitHub
2. Press the **period key (.)** on your keyboard
3. A web-based code editor will open in your browser

## Step 2: Create Your Own Branch

Think of a branch as your own workspace where you can make changes without affecting the main site.

1. Press **Ctrl + Shift + P** (Windows) or **Cmd + Shift + P** (Mac)
2. Type **"Create branch"** in the search box
3. Click **"Remote Repositories: Create Branch"**
4. Give your branch a simple name (e.g., "update-december-events" or "add-new-vendor")
5. Press **Enter**

## Step 3: Navigate to the Data Files

1. On the left sidebar, click to expand folders
2. Go to: **src** → **lib** → **data**
3. You'll see the data files you can edit

## Step 4: Edit the Data

### For .json files:

- Click on the file to open it
- Make your changes carefully
- Keep the structure the same (commas, brackets, quotes)

### For .md files:

- Click on the file to open it
- **Important:** Make sure the filename matches the `id` in the top section
- Edit the content below the metadata section

## Step 5: Save Your Changes

1. After editing, look at the left sidebar for a source control icon (looks like a branch)
2. Click it to see your changes
3. Type a brief message describing what you changed (e.g., "Updated vendor hours")
4. Click the **checkmark** button to commit your changes

## Step 6: Create a Pull Request

1. You'll see a notification to create a pull request
2. Click **"Create Pull Request"**
3. Add a description of what you changed
4. Click **"Create"**
5. Done! Your changes will be reviewed before going live
